---
title: Gatsby Commands
---

Here are the list of gatsby Commands

to create a new project with an interative shell asking for options

```js
gatsby new
```

after gatsby project is created, go to the root directory and run develop

```js
gatsby develop
```

to run the build and make the project ready for deployement

```js
gatsby build
```

to serve the production build of the project for testing

```js
gatsby serve
```
